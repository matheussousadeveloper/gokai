# Gokai: Sistema de Gestão de Alunos para Academias
## Sobre o Projeto ##
Gokai é um Sistema de Gestão de Alunos (SaaS) dedicado a simplificar o cadastro, o acompanhamento e a administração financeira de academias e dojos

## Funcionalidades Principais (MVP) ##
- Cadastro de Alunos: Inscrição rápida e organizada de novos membros.
- Gestão de Turmas: Associação de alunos a diferentes modalidades e horários.
- Controle de Mensalidades: Lançamento e acompanhamento básico de status de pagamento.
